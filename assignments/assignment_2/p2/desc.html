DESCRIPTION: this simple task for student to test the enviroment

INPUT: first line is number of test case, for n next line, each line describe 1 test case, which include 2 integer a(-2^32<=a<=2^31-1) and b(2^32<=b<=2^31-1)

OUTPUT: for each test case, print a + b, assume that result fit in regular integer type

SAMPLE:

input 

3
5 5
4 3
7 8

output

10
7
15
